Hello and thank you for installing this program.

Description:
This Web Application called Resume Forge is intended for students and other potential employees to create resumes.
The user is able to add and save details like jobs, job responsibilities, skill categories, skills, certifications, awards, and most importantly Google Gemini API Keys.

Requires an account

If you have trouble with this application run these commands:
npm init
npm install express cors sqlite3 bcrypt @google/genai
node .
